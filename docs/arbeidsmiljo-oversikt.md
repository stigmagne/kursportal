# Oversikt: Arbeidsmiljø-modulen i Kursportalen

Arbeidsmiljø-modulen er delt inn i to roller:
- **Team-medlem** (arbeider/medarbeider)
- **Team-leder** (leder)

---

## Tilgjengelig for BEGGE roller (arbeider + leder)

### Journalverktøy (5 stk) - kryptert

| Verktøy | Beskrivelse |
|:--------|:------------|
| **Daglig Sjekk-inn** | Kort daglig refleksjon over energinivå, trygghetsfølelse og hovedfølelse på jobb |
| **Trygghetsdagbok** | Loggfør situasjoner der du følte deg trygg eller utrygg - hva skjedde, hvem var involvert, hva hjalp |
| **Feedback-logg** | Hold oversikt over tilbakemeldinger du gir og mottar, med kontekst og refleksjon |
| **Konflikt-refleksjon** | Analyser konfliktsituasjoner - utløser, din rolle, underliggende behov, utfall |
| **Grense-tracker** | Følg med på om du holder grensene dine - sa du nei, jobbet du overtid, tok du pauser |

---

## Kun for ARBEIDER (team-member)

### Kurs (6 stk)

| Kurs | Fokus |
|:-----|:------|
| **Trygg på Jobb** | Forstå psykologisk trygghet og hvordan du kan bidra til trygghet på arbeidsplassen |
| **Min Plass i Teamet** | Finn din plass og verdi - forskjellen på tilhørighet og tilpasning |
| **Kommunikasjon på Jobb** | Aktiv lytting, tydelige beskjeder, vanskelige samtaler |
| **Sunne Grenser på Jobb** | Sette og holde grenser for tid, energi og arbeidsoppgaver - work-life balance |
| **Håndtere Konflikt** | Konstruktive måter å håndtere uenigheter med kollegaer |
| **Vekst og Mestring** | Utvikle growth mindset og håndtere utfordringer som læringsmuligheter |

### Vurdering (30 spørsmål, 6 dimensjoner)

| Dimensjon | Måler |
|:----------|:------|
| Psykologisk Trygghet | Tørre å si fra, stille spørsmål, ta risiko uten frykt |
| Tilhørighet | Inkludering, følelse av å være verdsatt i teamet |
| Kommunikasjon | Kvalitet på kommunikasjon med kollegaer og ledere |
| Grensesetting | Evne til å si nei, balanse mellom jobb og fritid |
| Konflikthåndtering | Håndtering av uenigheter og vanskelige situasjoner |
| Vekst og Mestring | Personlig og faglig utvikling på jobben |

---

## Kun for LEDER (team-leader)

### Kurs (6 stk)

| Kurs | Fokus |
|:-----|:------|
| **Lederen som Trygghetsskaper** | Skape et miljø der folk tør å feile og lære - modellere sårbarhet |
| **Inkluderende Ledelse** | Håndtere ubevisste fordommer, inkludere alle stemmer, mangfold |
| **Tilbakemeldingskultur** | Gi og motta feedback (SBI-modellen), 1:1-samtaler |
| **Delegering og Tillit** | Delegere ansvar og gi rom til vekst uten å miste oversikt |
| **Lederens Konflikthåndtering** | Meklerrollen, fasilitere løsning mellom parter |
| **Lederens Egenomsorg** | Håndtere stress, sette grenser, bygge støttenettverk |

### Vurdering (30 spørsmål, 6 dimensjoner)

| Dimensjon | Måler |
|:----------|:------|
| Skape Trygghet | Evne til å skape trygt miljø der folk tør å feile |
| Inkluderende Ledelse | Hvordan du inkluderer alle stemmer og skaper mangfold |
| Tilbakemeldingskultur | Hvordan du gir og mottar tilbakemeldinger |
| Delegering og Tillit | Evne til å delegere ansvar og vise tillit |
| Lederens Konflikthåndtering | Håndtering av konflikter i teamet |
| Lederens Egenomsorg | Hvordan du tar vare på deg selv som leder |

---

## Onboarding

- **Obligatorisk vurdering**: Førstegangsbrukere må fullføre vurderingen før kurstilgang
- **Re-vurdering**: Påminnelse hver 3. måned for å spore progresjon
- Kursanbefalinger baseres på vurderingsresultater per dimensjon
